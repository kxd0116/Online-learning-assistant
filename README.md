### 脚本功能说明：

1. **自动处理提示框**：当出现"当前课件学习时间已达到系统设置时间"的提示时，自动点击"确定"
2. **防挂机检测**：定期模拟用户活动（鼠标移动、键盘事件等），避免被系统检测为挂机
3. **智能监控**：监控DOM变化，捕获动态生成的提示框
4. **弹窗拦截**：重写alert/confirm方法，自动处理特定提示
5. **自动切换**：达到学时后自动切换到下一个课件
    

### 注意事项：

1. 脚本只会在匹配的URL上运行（你的学习网站）
2. 脚本会模拟用户活动，但频率较低，不会影响性能
3. 如果网站更新了提示框的结构，可能需要调整脚本
4. 请遵守学习平台的使用条款，合理使用

### 安装地址

### 方法一：使用 Tampermonkey 扩展（推荐）

1. **安装 Tampermonkey 扩展**
    - 在 Edge 浏览器中打开扩展商店
    - 搜索 "Tampermonkey" 并安装
2. **安装脚本**
    - 点击 Tampermonkey 图标
    - 选择"添加新脚本"
    - 删除默认代码，粘贴上面的完整脚本
    - 按 `Ctrl+S` 保存
3. **启用脚本**
    - 访问你的学习网站
    - Tampermonkey 会自动检测并启用脚本
    - 页面右上角会显示"✅ 学习助手已启用"的提示

### 方法二：使用 Edge 开发者工具

1. 按 `F12` 打开开发者工具
2. 切换到"控制台"选项卡
3. 粘贴脚本代码并回车执行（注意：页面刷新后需要重新执行）
